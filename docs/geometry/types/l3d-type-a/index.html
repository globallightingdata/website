Round Recessed Luminaires (Type A)

Round recessed luminaires derive from the type "Cylinder" with no additional fields.

ParameterOccuranceDescriptionDefault value
DiameterrequiredTotal diameter of the luminaire.200 mm
TopDiameteroptionalDiameter of the upper surface. If this value is present and different from Diameter, it skews the shape of the housing.200 mm
HeightrequiredTotal height of the housing.200 mm
LuminousDiameterrequiredDiameter of the luminous volume or surface300 mm
LuminousHeightoptionalTotal height of the luminous volume. Values greater than one will extrude out from the luminaire. One will indicate a flat surface.1 mm
