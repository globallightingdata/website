# GLDF Website

Website of the *Global Lighting Data Format* ➔ [gldf.io](https://gldf.io)

## Requirements

- Node.js ➔ [nodejs.org](https://nodejs.org)
- Yarn ➔ [yarnpkg.com](https://yarnpkg.com)

## Installation

```console
yarn install
```

## Local Development

```console
yarn start
```

## Build

```console
yarn build
```
