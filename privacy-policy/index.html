2. General Information on Data Processing​

As a rule, we process personal data only to the extent necessary to provide a functional website and to ensure the stability and security of the website. The legal basis for such processing is Art. 6(1)(f) GDPR. Where further processing takes place in individual cases, it will only be carried out on the basis of a legal permission or your consent. Art. 13 GDPR requires transparency regarding the purposes of processing, legal bases, recipients, and any transfers to third countries.

3. Provision of the Website and Hosting via GitHub Pages​

This website is hosted via GitHub Pages, a service provided by GitHub, Inc. GitHub states that when a GitHub Pages website is visited, the visitor’s IP address is logged and stored for security purposes, regardless of whether the visitor is signed in to GitHub or not.

In this context, the following technical data may in particular be processed:

  • IP address
  • date and time of the request
  • requested file or URL
  • amount of data transferred
  • notification of successful retrieval
  • browser type and browser version
  • operating system
  • referrer URL

The processing is carried out for the purpose of providing the website, ensuring its technical stability, and maintaining the security of the service. The legal basis is Art. 6(1)(f) GDPR. Our legitimate interest lies in the secure and reliable delivery of the website.

4. External Images Hosted by Imgur​

Some images on this website are loaded from external servers operated by Imgur.

When you visit a page containing such an image, your browser retrieves the image file directly from Imgur’s servers. In this process, technical access data may be transmitted to Imgur, in particular your IP address and browser-related request data. Depending on your browser and configuration, the referrer URL may also be transmitted.

This processing is carried out for the purpose of displaying the embedded images and ensuring the technical delivery of the content. The legal basis is Art. 6(1)(f) GDPR. Our legitimate interest lies in the efficient presentation of the website and its visual content.

This website uses Algolia to provide site search functionality.

When you use the search function, your search terms and technical request data may be processed by Algolia in order to provide search results and maintain the search service. The legal basis for the use of the search function is Art. 6(1)(f) GDPR. Our legitimate interest lies in providing an efficient and user-friendly search function for the content of this website.

Recipients of personal data processed in connection with the use of this website may include:

  • GitHub, whose infrastructure is used to host and deliver this website via GitHub Pages
